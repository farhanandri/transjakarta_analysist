# Deskripsi
This project performs an end-to-end analysis of TransJakarta passenger transactions, starting from raw data processing, data warehouse development using a fact–dimension schema, and culminating in an interactive dashboard. The analysis uncovers demand patterns, peak-hour behavior, high-volume corridors and stops, and weekday–weekend travel differences. Insights generated from the dashboard support operational recommendations that enhance service performance and promote sustainable urban mobility
